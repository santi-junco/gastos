# gastos
Requerimientos del proyecto
pantalla de inicio con formulario de inicio de sesion o registro de usuario
una ves que se inicia sesion se accede a la pagina de inicio donde se listan los gastos diarios 
gastos diarios
	fecha
	monto
	descripcion
se debe poder acceder a una seccion donde se ven los gastos fijo y los ingresos
gastos fijos
	fecha 
	monto 
	descripcion
ingresos
	fecha
	monto
	descripcion
se debe poder crear, editar y eliminar gastos diarios, fijos e ingresos
debe haber un historial de los gastos del mes anterior
al final del mes se debe mostrar un balance de los gastos 
se debe mostrar un totalizador de montos en cada una de las tablas
